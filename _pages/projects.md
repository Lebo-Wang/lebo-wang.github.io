Spatial Graph Convolution Network for Accurate fMRI Data Classification
======
* Spatial graph convolution on ROI-based fMRI data for mental disorder classification
* K-nearest neighbors graph for convolutions based on group functional connectivity (FC)
* Asymmetric edge function for co-activation pattern modeling of fMRI data
* Topological property comparison between k-NN graph and the threshold FC

Convolution Neural Network (CNN) for interpretable fMRI Data Analysis
======
* Traditional CNN on ROI-based fMRI data for co-activation pattern extraction
* Recurrent neural networks for the temporal dynamics modeling of fMRI data
* Salient region visualization based on soft-attention, occlusion and gradient method
* Different data preprocessing methods and dynamic data augmentation for training
